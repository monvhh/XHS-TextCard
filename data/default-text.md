# XHS-TextCard 使用指南

> **XHS-TextCard** 是一款专为小红书创作者打造的智能排版工具。它能将您的深度长文一键转化为排版精美的系列卡片，助你打造独一无二的个人 IP 视觉风格。

本指南将帮助您快速上手，掌握如何利用本工具创作出极具吸引力的文字内容。

🌐 **官方在线版**：https://xhs-textcard.site/

📖 **使用指南**：https://xhs-textcard.site/guide.html

## 🌟 核心功能概览

**智能自动分页** - 输入文字后，系统会自动根据内容长度将其拆分为多张 3:4 比例的高清图片。

**实时视觉反馈** - 编辑文字或调整样式时，预览区会实时呈现最终生成的图片效果，所见即所得。

**Markdown 深度支持** - 支持 # 标题、**加粗**、==高亮==、引用、列表、LaTeX 公式、代码高亮等多种语法，让文字更具层次。

## 📏 完美的排版与分页控制

系统内置了精密的排版引擎，能够确保文字在每一张卡片上的分布都科学、美观。

### 创作小技巧：
1. **自动分页**：系统会优先在段落之间进行分页。建议使用两个回车分隔段落，这样生成的卡片阅读体验最佳。
2. **强制分页**：如果您想在特定的地方切断并开启新的一页，只需输入 `---`（三个短横线），系统会立即在此处分页。

## 🆔 打造个人品牌 IP

**专属品牌签名**
内置现代胶囊、终端代码、优雅衬线等多种签名设计，让每一份内容都带有你的专属印记。

**版权防盗水印**
自由开启背景水印，可精细调节文字透明度，在不影响阅读的前提下保护原创内容。

**高清批量导出**
排版完成后，点击「打包下载全部」按钮，系统将自动生成 1242x1656 高清图片（小红书最佳尺寸），并打包为 ZIP 压缩包供您一键保存。
